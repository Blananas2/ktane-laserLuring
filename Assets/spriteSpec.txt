Cat Sprites:
    Number is via the following formula
    (10xB)+A
    where B is the Breed:
        0 = Orange
        1 = Black (mostly undrawn)
        2 = White (mostly undrawn)
    where A is the Action:
        0 = Sitting
        1-8 = Walk Cycle [ascending frame order]
        9 = Pouncing (undrawn)
{May adjust formula later; will likely add more breeds}

Item Sprites are in reading order in the manual
Note that care needs to be taken to go from original coord => item bc of the extra step

Furnature Sprites:
0 = Couch
1 = Shelf
2 = Bookcase
3 = Piano (undrawn)

Other Sprites:
0 = Cursor
1 = Collar Sitting
2 = Collar Walk Cycle
3 = Collar Pouncing (undrawn)
4 = Laser (undrawn)
5 = Cat Smile

Slots:
0-2 = Cats
3-5 = Collars
6-8 = Smiles
9-18 = Furnature
19-23 = Items
24-35 = Cursors

Layers:
0 = Background
1 = Furnature
2 = Items
3,4,5 = Cats
6 = Collars, Smiles
7 = Cursors
100 = Debug
